i3-wm
A majority of the shortcuts are mapped from Windows 10. This is simply to make the workflow easier to adapt to.

Close Window
    Super + Q

Search
    Super + D

End session / log out
    Super + Shift + E

Refresh i3
    Super + Shift + R

Move focused window to another workspace
    Super + Shift + 1-9

Change workspaces
    Super + 1-9

sysZ Main Interface
    Super + i

Lock workstation
    Super + L

Display Settings
    Super + P

Open File Manager (Thunar)
    Super + E

Open Terminal (Alacritty)
    Super + Enter

Cycle thru active workspace's containers
    Alt+Tab

Overview of all open containers
    Super + Tab

Cycle thru workspaces
    Control + Super + (<) LEFT_ARROW_KEY or RIGHT_ARROW_KEY (>)
